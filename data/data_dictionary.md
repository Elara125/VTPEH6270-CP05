This dataset contains COVID-19 testing data from New York State.

- test_date: Date of COVID-19 test (Date)
- geo_name: Name of county/region/state (Categorical)
- geo_level: Level of geography (STATEWIDE, REGION, COUNTY)
- new_positives: Number of new positive cases (Discrete)
- cum_positives: Cumulative positive cases (Discrete)
- tests_performed: Number of tests performed (Discrete)
- cum_tests_performed: Cumulative tests performed (Discrete)
- pct_positive: Percent of positive tests (Continuous)
