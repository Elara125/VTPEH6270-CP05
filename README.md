# VTPHE 6270 - Check Point 05

## Project Overview
This repository contains data, code, and outputs for CP02–CP04 assignments in VTPHE 6270. The project focuses on data exploration, comparative analysis, and simulation using public health datasets.

## Author
Jiaru Wei  
Master of Public Health (MPH), Cornell University  

## Research Objective
The objective of this project is to explore patterns in public health data and examine relationships between key variables (e.g., obesity and physical inactivity), as well as simulate potential outcomes based on observed trends.

## Data Sources
- CDC PLACES dataset  
- New York State COVID-19 Testing dataset  

## Repository Structure

- `data/raw/` : Original datasets used for analysis  
- `data/processed/` : Cleaned datasets (if applicable)  
- `scripts/` : R Markdown files for data exploration, analysis, and simulation  
- `outputs/figures/` : Generated figures  
- `outputs/reports/` : Final PDF reports  

## Reproducibility

To reproduce results:

1. Open RStudio  
2. Run the `.Rmd` files in the `scripts/` folder  
3. Required packages:
   - ggplot2  
   - dplyr  
   - tidyr  

## Outputs

- CP02: Data exploration report and visualizations  
- CP03: Comparative analysis results  
- CP04: Simulation outputs and plots  

All outputs are available in the `outputs/` folder.
